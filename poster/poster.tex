	  \headerbox{MATERIALES}{name=materiales,column=1,row=0,span=2}{
	  	\begin{itemize}
	  \item 2 botellas de dos litros.
	  \item Un tapón de corcho.
	  \item Una aguja para inflar pelotas.
	  \item Una bomba de aire.
	  \item Un cartón de leche.
	  \item Un marcador.
	  \item Un cúter.
	  \item Una pistola de silicona caliente.
	  \item Cinta adhesiva.
	  
	  	\end{itemize}	
	  }
	  \headerbox{PASOS A SEGUIR}{name=pasos,below=materiales,column=1,span=2}{
	  	\begin{enumerate}
	  	\item Cortar con el cúter el cono de una de las botellas.
	  	
	  	\item Pegar con silicona caliente a la parte inferior de la otra. 
	  	
	  	\item La parte sobrante la vamos a utilizar de soporte, así que hacemos un agujero por el que después meteremos la manguera de la bomba de aire, más o menos en la mitad de éste.
	  	
	  	\item Lo siguiente que tenemos que hacer son las alas de nuestro cohete. Para ello, trazamos con el marcador una línea de esquina a esquina por una de las caras y cortar por aquí con el cúter. 
	  	
	  	\item Pegar a los laterales de nuestro cohete las alas con silicona caliente y fijar bien con cinta adhesiva.
	  	
	  	\item Introducir la aguja con fuerza por el tapón de corcho, de manera que salga por el otro lado. 
	  	
	  	\item Meter la manguera por el agujero que hemos hecho anteriormente en el soporte de lanzamiento, el cual tendrá que estar lleno de arena o piedras para que se quede fijo en el suelo.
	  	
	  	\item Tan sólo echar unos 400 ml de agua en el cohete casero y dar presión con la bomba hasta que salga despedido.
	  	
	  	\end{enumerate}
	  	
	  }
	  
	  \headerbox{OTRAS RECOMENDACIONES}{name=otras,below=pasos,column=1,span=2}{
	  Aparte de lo que hemos comentado de como llevar a cabo un cohete impulsado por agua, también se pueden hacer otras muchas modificaciones:
	  \begin{itemize}
	  	\item Colocación de varias fases para que se impulse mucho más alto.
	  	\item Elaboración de una estructura de lanzamiento mejorada, que permita la compresión del gas y mediante una palanca soltar el cohete para aumentar el realismo.
	  	\item Fabricación de un paracaídas para frenar el impacto, con un temporizador que salte en un momento determinado de la caída. 
	  	\item Elementos decorativos como pintarle las alas, pintar y elaborar una estructura donde irá el cohete.
	  \end{itemize} 
	  	
	  	}
	  	
	   \headerbox{PRECAUCIONES}{name=precauciones,below=otras,column=1,span=2}{
	   	Hay que tener en cuenta algunas recomendaciones a la hora de llevar a cabo la construcción del cohete:
	   	\begin{itemize}
	   		\item Debido a que sale disparado es conveniente que se utilice protección y que se mantenga una distancia de seguridad.
	   		\item Como se va a meter presión en la botella hay que tener precaución con que no estalle y que si estalla se esté lo más lejos y protegido posible.
	   		\item A la hora de tratar con la silicona y el cúter hay que tener especialmente cuidado pues son elementos que pueden causar lesiones cutáneas.
	   	\end{itemize}
